Avelune

Privacy Policy

Last updated: April 25, 2026

Overview

Avelune is a personal planning app designed to help users create time blocks, protect focus time, and keep the day visible. Avelune does not require an account to use its core functionality.

Information Stored On Device

Avelune stores planner blocks, focus session history, reminder preferences, and widget snapshot data on the user’s device. This information is used to power the app’s local planning, focus, insight, and widget experiences.

Calendar Access

If the user grants calendar access, Avelune reads calendar events using Apple EventKit so the app can show a read-only calendar overlay beside the user’s manual plan. Avelune does not write events back to the user’s calendar.

Notification Access

If the user grants notification access, Avelune uses Apple UserNotifications to schedule daily planning reminders and focus session completion alerts requested by the user.

Purchases

Avelune Pro subscriptions are offered using Apple StoreKit. Purchase processing, billing, and subscription management are handled by Apple. Avelune does not receive full payment card information.

Widgets And Shared Data

If the user enables the widget extension, Avelune stores a lightweight planning snapshot in the shared app group so Home Screen widgets can display the next block and daily summary.

Third-Party And Platform Services

Avelune relies on Apple platform frameworks such as StoreKit, EventKit, UserNotifications, WidgetKit, App Groups, and SwiftData. Core app functionality does not require an external account or analytics service.

Regional Availability

Avelune is intended to function consistently across all supported regions. No region-specific differences in privacy handling are intentionally applied in the current version.
